USAGE:

delta_SA_ref = gsw_delta_SA_ref(p,long,lat)

DESCRIPTION:

Calculates the Absolute Salinity anomaly.
Click for a more detailed description of the 
Absolute Salinity anomaly reference value.

INPUT:

p     =  sea pressure                                           [ dbar ]
         ( i.e. absolute pressure - 10.1325 dbar )
long  =  longitude in decimal degrees                     [ 0 ... +360 ]
                                                    or [ -180 ... +180 ]
lat   =  latitude in decimal degrees north               [ -90 ... +90 ]
p, long & lat need to have the same dimensions.

OUTPUT:

delta_SA_ref  =  Absolute Salinity anomaly reference            [ g/kg ]

EXAMPLE:

p =    [   10;   50;  125;  250;  600; 1000;]
lat =  [    4;    4;    4;    4;    4;    4;]
long = [  188;  188;  188;  188;  188;  188;]
[delta_SA_ref, in_ocean] = gsw_delta_SA_ref(p,long,lat)
delta_SA_ref =              in_ocean =
  0.000167785807437       1
  0.000268675908040       1
  0.000665539507353       1
  0.002694303422857       1
  0.005626663909471       1
  0.009396653216531       1
